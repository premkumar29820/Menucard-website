# Menu Card Website

This is a simple **Menu Card Website** built using **HTML** and **CSS**. The project displays a restaurant-style menu in a clean and visually appealing layout. It was created to practice basic web development concepts, including page structure, styling, and content organization.

## Features

- Single-page menu card design
- Clean and user-friendly layout
- Organized food and beverage sections
- Basic CSS styling for an attractive appearance
- Responsive and easy-to-read design

## Technologies Used
- HTML
- CSS

## Project Structure

## Purpose
This project was developed to strengthen my understanding of HTML and CSS by creating a simple and visually appealing menu card website.

## Preview
The website presents menu items in a structured format, making it easy for users to browse food and beverage options.
